# Firestrike ‚Äì First Person Shooter (Unity)

A level-based First Person Shooter (FPS) survival game built in Unity. Designed as an action-packed combat experience where players fight through escalating waves of enemies while managing ammo, health, and powerful weapons. This project showcases core FPS systems such as player movement, shooting mechanics, enemy AI, UI, level design, and a score-based survival mode.

---

## üéÆ Game Overview
**Firestrike** is a fast-paced FPS game where you play as a lone soldier fighting waves of enemies in a sci-fi themed environment. With multiple weapons, dynamic lighting, collectibles, and a polished HUD, the game tests your precision, movement, and survival strategy.  
Survive as long as possible, defeat enemies, collect ammo/health, and push for higher scores.

---

## Key Features
- **Smooth FPS Controller**  
  - Walking, sprinting, jumping, and mouse-based aiming
- **Weapon System**  
  - Multiple gun types: *Pistol, Rifle, Shotgun*  
  - Reloading, ammo management & weapon switching  
  - Shooting VFX and impact effects  
- **Enemy AI**  
  - Patrol, chase & attack behaviors  
  - Increasing difficulty with every wave  
  - Wave progression system  
- **Collectibles**  
  - Ammo packs  
  - Health packs  
  - Score boosters  
- **Level Design**  
  - Multi-level environments  
  - Sci-fi battleground atmosphere  
- **User Interface (UI)**  
  - Health, ammo, and score HUD  
  - Pause, Start, and Game Over menus  
- **Boss Battle**  
  - Unique boss with special attack patterns  
- **Audio System**  
  - Gun sounds, enemy sounds & environmental ambiance  

---

## How to Play

| Action | Control |
|-------|---------|
| Move | **W A S D** |
| Aim | **Mouse** |
| Shoot | **Left Click** |
| Reload | **R** |
| Switch Weapons | **1, 2, 3** |
| Jump | **Spacebar** |
| Pause Menu | **Esc** |
